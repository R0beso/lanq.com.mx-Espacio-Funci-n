Aprendiendo java

El objetivo principal de la creación del siguiente software es exponer funcionalidades que se
puede desarrollar con Java en un solo proyecto cliente-servidor, funcionalidades como:

• Uso de redes: Implementa comunicación utilizando protocolos como HTTPS con APIs con Spring Boot.

• Interfaz gráfica: Incluye una simple GUI interactiva utilizando herramientas como Swing,
y bibliotecas externas para experiencias de usuario de manera cómoda.

• Orientación a objetos: Utiliza clases, objetos, encapsulación y abstracción.

• Manejo de excepciones: Cuenta con mecanismos para la gestión de errores mediante try-catch.

• Bases de datos: Utiliza JDBC y frameworks ORM Hibernate para conectar y manejar datos almacenados en bases relacionales SQLite.

• Gestión de archivos: Permite leer en archivos en la parte del servidor, procesar archivos y realizar operaciones de entrada/salida.

• Seguridad: Asegura el sistema mediante cifrado técnicas como SSL/TLS.

• Multitarea y concurrencia: Integra hilos (Thread) y ejecutores para realizar tareas simultáneas, optimizando el rendimiento.
